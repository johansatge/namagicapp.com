Overview

Namagic is a batch renaming software that offers you to work with your files in a brand new way. Its aim is to provide a flexible solution, wether it be used to rename a few vacations pictures, or work with hundred of files and several operations.

Adding files

Drag and drop the files you want to work with in the application window, or select them in the files dialog.
You can add as many files as you want, and all file types are supported.

Selecting text

Select the part of the filenames you want by using one of the providden choices: free text, list of characters, range of text...

Configuring actions

Create one or more action(s) to modify the part of the filename that you previously selected. You may remove the selection, append new text, a sequence of digits or the creation date of the file...

Applying the new names

Apply your operations to rename the files.
Select your destination folder - on most cases, the same as your source files - and let the magic happen !
